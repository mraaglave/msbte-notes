Operating System - 5th Semester MSBTE

The following outline includes unit-wise outcomes, topics, and subtopics for the Operating System subject in the 5th semester of MSBTE.

Unit I: Overview of Operating System

Unit Outcomes (UOs):

  • Explain the functioning of various components of OS.
  • Describe characteristics of different types of operating systems.
  • Identify a suitable type of OS for specific applications.
  • Execute basic commands on the command line.
  • Start, stop, and restart services in Linux.

Topics and Sub-topics:

  • Concept of Operating System, Components, Program Management, Resource Management, Security and Protection.
  • Different Views of OS: User View, System View.
  • Types of Operating Systems: Batch, Multi-programmed, Time-shared, Multiprocessor, Distributed, Real-time, Mobile OS (Android, iOS).
  • Command Line OS: DOS, UNIX; GUI-based OS: Windows, Linux.

Unit II: Services and Components of Operating System

Unit Outcomes (UOs):

  • Explain the use of system calls in OS.
  • Describe the process of resource management in OS.
  • Use specific OS tools for managing users and security policies.

Topics and Sub-topics:

  • Different Services provided by Operating Systems.
  • System Calls: Concept, Types.
  • OS Components: Process Management, Main Memory Management, File Management, I/O System Management, Secondary Storage Management.

Unit III: Process Management

Unit Outcomes (UOs):

  • Explain the functions performed in various process states.
  • Describe the process control block (PCB) and its components.

Topics and Sub-topics:

  • Process: States, Process Control Block (PCB).
  • Process Scheduling: Queues, Schedulers, Context Switch.
  • Inter-process Communication (IPC): Shared Memory, Message Passing.
  • Threads and Multithreading Models: Many-to-One, One-to-One, Many-to-Many.

Unit IV: CPU Scheduling and Algorithms

Unit Outcomes (UOs):

  • Justify the need and objective of job scheduling.
  • Explain the procedure for CPU allocation using various scheduling algorithms.

Topics and Sub-topics:

  • Scheduling Types and Objectives.
  • Scheduling Algorithms: First Come First Served (FCFS), Shortest Job First (SJF), Round Robin (RR), Priority Scheduling, Multilevel Queue.
  • Deadlock: System Model, Necessary Conditions, Handling Techniques (Prevention, Avoidance).

Unit V: Memory Management

Unit Outcomes (UOs):

  • Describe various memory management functions and techniques.
  • Explain the working of page replacement algorithms.

Topics and Sub-topics:

  • Basic Memory Management: Partitioning (Fixed, Variable), Free Space Management (Bitmap, Linked List).
  • Virtual Memory: Paging, Segmentation, Fragmentation, Page Fault.
  • Page Replacement Algorithms: FIFO, LRU, Optimal.

Unit VI: File Management

Unit Outcomes (UOs):

  • Describe the structure of a file system and access methods.
  • Explain directory structure and file access permissions.

Topics and Sub-topics:

  • File Concepts: Attributes, Operations, Types, and Structure.
  • File Access Methods: Sequential, Direct, Indexed.
  • Directory Structures: Single-level, Two-level, Tree-structured.
  • Disk Structure: Physical and Logical, RAID Levels (0 to 6).

By the end of this course, students will have a comprehensive understanding of the functions, management techniques, and essential tools in operating systems.
